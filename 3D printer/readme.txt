This project is a scrappy built 3D printer. It has been bought as a kit (Tevo Tarantula) in 2018 and since has gone through many modifications including:
-new hotend (E3D V6) with custom mounting
-new PSU
-second Z-axis
-changed A4988 drivers to TMC2130
-added bed leveling sensor BLTouch
-added supports
-added bed leveling wheels
